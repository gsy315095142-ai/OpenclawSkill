# 联动规则备忘

> 当修改数值时，必须同步检查以下联动项

---

## 一、积分联动

| 触发条件 | 需同步更新 | 计算公式 |
|---------|----------|---------|
| 积分规则调整 | 晋级规则场次数 | 见下方 |

**计算公式：**
```
100%胜率升段场数 = 500 / 同段位赢积分
50%胜率升段场数 = 500 / (同段位赢积分 - 同段位输扣分) × 2
```

**当前值：**
- 同段位赢积分 = +50
- 同段位输扣分 = -5
- 100%胜率 = 500/50 = **10 场**
- 50%胜率 = 500/(50-5)×2 = **22 场**
