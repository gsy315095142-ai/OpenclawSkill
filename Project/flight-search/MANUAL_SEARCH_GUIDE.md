# 🛫 手动航班搜索操作指南

## 📋 快速操作步骤

### 1️⃣ 安装脚本（2分钟）
1. 安装 **Tampermonkey** 扩展
2. 创建新脚本，复制 `manual_flight_search.js` 内容
3. 保存并启用

### 2️⃣ 访问网站（1分钟）
打开以下任一网站：
- **携程**：https://flight.ctrip.com
- **去哪儿**：https://flight.qunar.com

### 3️⃣ 执行搜索（3分钟）
1. 按 **F12** 打开控制台
2. 输入：`startFlightSearch()`
3. 脚本自动填充：厦门→北京，明天，商务舱
4. 手动点击**搜索**按钮

### 4️⃣ 提取数据（1分钟）
搜索结果出来后，在控制台输入：`extractFlightData()`

## 🎯 预期结果

✅ **至少10个航班**的真实数据  
✅ **自动价格排序**，找出最便宜  
✅ **生成HTML报告**，美观展示  
✅ **下载本地文件**，随时查看  

## 💡 关键提示

- ⏰ **早上航班**：关注07:00-12:00起飞的航班
- 💺 **商务舱**：确保选择Business Class
- 🏙️ **两个机场**：北京首都(PEK)和大兴(PKX)
- 📱 **多平台**：建议搜索2-3个网站获得全面数据

## 🔧 故障排除

如果脚本无法运行：
1. 手动输入搜索条件
2. 搜索完成后运行：`extractFlightData()`
3. 或者把航班信息发给我，我来处理
