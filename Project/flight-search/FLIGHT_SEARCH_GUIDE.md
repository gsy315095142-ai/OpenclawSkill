# 🛫 厦门北京航班数据真实搜索指南

## 📋 操作步骤

### 1. 安装浏览器扩展
- 安装 **Tampermonkey** 扩展（Chrome/Firefox/Edge）
- 地址：https://www.tampermonkey.net/

### 2. 安装脚本
1. 打开 Tampermonkey 控制面板
2. 点击「添加新脚本」
3. 将 `flight_data_extractor.js` 的内容复制进去
4. 保存脚本

### 3. 访问机票搜索网站
打开以下任一网站：
- **携程**：https://flight.ctrip.com
- **去哪儿**：https://flight.qunar.com  
- **Skyscanner**：https://www.skyscanner.com

### 4. 自动填充搜索条件
在网页控制台中运行：
```javascript
autoFillSearchForm()
```

### 5. 手动完成搜索
- 选择**商务舱**
- 设置日期为**明天** (2026-02-06)
- 点击**搜索**按钮

### 6. 提取航班数据
搜索完成后，在控制台运行：
```javascript
extractFlightData()
```

## 🎯 脚本功能

✅ **自动填充**：厦门 → 北京，明天日期  
✅ **数据提取**：航班号、时间、价格、航空公司  
✅ **智能识别**：适配多个机票网站  
✅ **HTML生成**：自动生成美观的结果页面  
✅ **价格排序**：自动找出最佳性价比  

## 📊 预期结果

运行成功后会：

## 🔧 备用方案

如果脚本无法运行，你可以：

### 手动数据收集
1. 访问上述机票网站
2. 手动搜索厦门→北京，明天，商务舱
3. 记录下至少10个航班的信息：
   - 航空公司
   - 航班号
   - 起飞时间
   - 到达时间
   - 价格
   - 机型

## 💡 提示

- **多平台对比**：建议在2-3个平台搜索，获得更全面的数据
- **早中晚班次**：确保包含不同时段的航班
- **两个北京机场**：首都机场(PEK)和大兴机场(PKX)
- **价格变化**：机票价格实时变动，搜索速度要快
